# Portfolio - Franco Guanes

Sitio web portfolio personal, construido con HTML, CSS y JavaScript puro (sin frameworks ni dependencias).

## Estructura

```
├── index.html      # Contenido y estructura del sitio
├── style.css       # Estilos
├── script.js       # Menú móvil y año dinámico del footer
└── README.md
```

## Cómo verlo localmente

Solo abrí `index.html` en el navegador (doble click), o usá una extensión como "Live Server" en VS Code.

## Despliegue en GitHub Pages (gratis)

1. Creá un repositorio nuevo en GitHub (por ejemplo `portfolio`).
2. Subí estos archivos al repositorio (ver comandos en la guía que te dio el asistente).
3. En GitHub, ir a **Settings → Pages**.
4. En "Source" elegí la rama `main` y la carpeta `/ (root)`.
5. Guardá. En un par de minutos el sitio queda publicado en:
   `https://<tu-usuario-de-github>.github.io/<nombre-del-repo>/`
